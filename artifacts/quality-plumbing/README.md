# Quality Plumbing Service

Professional plumbing, tile installation, bathroom renovation and emergency repairs in Chhatrapati Sambhajinagar, Maharashtra.

## Project Structure

```
/
├── index.html          ← Main website (open this to view locally)
├── style.css           ← All styles
├── script.js           ← All interactivity
├── assets/
│   └── images/         ← Project photos
└── README.md
```

## How to Use

**Locally:** Open `index.html` in any browser. No installation required.

**Deploy on Netlify / Vercel / GitHub Pages:** Push to a repository and deploy from the root folder. No build command needed — set build command to blank and publish directory to `/` (root).
